# 마음온 일기 (MaumOn Diary)

하루의 감정을 기록하며  
사용자가 스스로의 상태를 인식하고 정리할 수 있도록 돕는  
감정 중심 일기 웹 서비스 프로젝트입니다

## 프로젝트 개요
마음온 일기는 단순한 기록 기능보다  
사용자가 감정을 인식하고 정리하는 흐름에 집중한 프로젝트입니다.


## 기획 의도
감정은 쌓이지만 이를 표현하거나 정리할 기회는 많지 않다고 느꼈습니다.  
이 프로젝트는 꾸준함을 강요하지 않고,  
필요할 때 부담 없이 사용할 수 있는 감정 기록 웹 서비스를 목표로 합니다.

## 주요 기능
- 하루의 감정 상태를 중심으로 한 일기 작성
- 최소한의 입력으로 감정 기록이 가능한 UI
- 감정에 집중할 수 있는 차분한 화면 구성

## 기술 스택
- **Frontend**
  - React
  - Vite
  - TypeScript
- **Styling**
  - Tailwind CSS
- **State / Data**
  - React Hooks
  - LocalStorage
- **Tooling**
  - ESLint
  - Prettier
- **Version Control**
  - Git / GitHub
